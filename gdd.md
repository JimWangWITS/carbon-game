# 《低碳霸主：Low-Carbon Tycoon》完整遊戲設計文件（GDD）

---

# 1. 遊戲定位與核心體驗

## 遊戲類型

回合制企業競爭策略 × 地產建設 × 碳費管理 × 半塔防（排放怪獸）。

## 核心體驗

* 企業擴張競爭（與 3 家 NPC 搶市場、搶土地）
* 感受碳費制度帶來的壓力（排放越多成本越高）
* 經營策略抉擇（盈利 vs 減碳）
* 排放怪獸視覺化城市危機
* 雙市場：國內碳額度可交易／國外碳權只能自用
* 最多碳費淘汰 → 最高收益獲勝

---

# 2. 世界觀與背景敘事

在不遠的未來，各大企業在城市中快速擴張，但氣候危機迫在眉睫，政府全面啟動「碳費制度」，利用經濟誘因驅使企業轉型低碳經營。

玩家作為企業集團 CEO，與三家 NPC 企業在城市中競爭版圖、資源與市場。排放越多，城市上空的「排放怪獸」成長越快；若完全成型並踏入城市中心，全體玩家立即敗北。

---

# 3. 遊戲目標與勝負條件

## 主要目標

在不讓城市被排放怪獸摧毀的前提下：

1. **避免成為碳費最高者（直接淘汰）**
2. **存活玩家中，累積收益最高者獲勝**

## 敗北條件

* 個人敗北：本局結束時計算碳費最高者立即出局。
* 全體敗北：排放怪獸抵達城市中心（污染度 100%）。

---

# 4. 回合流程

每回合代表一年，由以下三階段組成：

## 階段一：玩家行動

每位玩家依序執行行動，包括：

* 購買土地
* 建造工廠（電廠/製造業/燃氣）
* 工廠升級
* 調整產量
* 購買國內額度
* 購買國際碳權
* 出售國內額度
* 提交自主減量計畫（中後期）

## 階段二：NPC 行動

NPC 依照其 AI 性格行動，包括：擴張、減碳、升級、買賣碳額度等。

## 階段三：年度結算

* 計算排放量
* 城市總排放 → 排放怪獸成長
* 扣除免徵額度（25,000）
* 行業係數套用（0.2～1.0）
* 工廠等級係數（50/100/300）
* 國內額度扣減（1 → 1.2）
* 國外碳權扣減（1 → 1）
* 碳費結算
* 收入計算
* 查核事件（每 3 回合）
* 第 6 回合起套用 CBAM 境外稅

---

# 5. 地圖與土地系統

## 地圖配置

* 隨機生成 20～30 塊土地
* 每塊地有屬性與成本差異

## 土地屬性

* 🌿 **綠電網區**：間接排放減少 50%
* 🧪 **高製程效率區**：直接排放減少 20%
* 🚢 **出口加工區**：行業係數影響（0.2→1.0）
* 🧱 **基本區**：無額外 buff
* ⚠ **高排放區**：排放增加 20%

---

# 6. 工廠系統

## 工廠類型

| 類型     | 收益 | 排放 | 特性               |
| ------ | -- | -- | ---------------- |
| 燃煤電廠   | 高  | 高  | 可轉型燃氣或綠能         |
| 燃氣電廠   | 中  | 中  | 升級後排放大幅降低        |
| 綠能電廠   | 低  | 很低 | 昂貴但後期超強          |
| 製造業    | 中  | 中  | 產量可調             |
| 高科技製造業 | 高  | 低  | 升級成本高            |
| 燃氣供應業  | 中  | 中  | 出口需求高（受 CBAM 影響） |

## 工廠等級（碳費公式核心）

| 等級       | 費率  | 效果      |
| -------- | --- | ------- |
| Lv1 一般   | 300 | 初始狀態    |
| Lv2 技術標竿 | 100 | 排放 -20% |
| Lv3 行業削減 | 50  | 排放 -40% |

# 7. 排放與碳費公式

## 年度排放公式

```
TotalEmission = Σ(直接排放 × 地塊係數 × 工廠等級係數) + 間接排放
```

## 收費排放量

```
Chargeable = (TotalEmission − 25,000) × 行業係數
```

## 碳權扣減

```
DomesticDeduction = min(DomesticCredit × 1.2, Chargeable × 10%)
InternationalDeduction = min(IntlCredit × 1.0, Chargeable × 5%)
```

## 最終碳費

```
CarbonFee = (Chargeable − DomesticDeduction − IntlDeduction) × FactoryRate
```

# 8. 排放怪獸（塔防系統）

排放怪獸代表城市污染壓力條，會逐年成長。

## 成長公式

```
MonsterGrowth += 全體排放量 / 污染門檻
```

## 階段事件

* 70%：怪獸狂暴，排放加成 ×1.2
* 90%：查核加倍、CBAM 恐慌
* 100%：怪獸抵達城市 → 全體玩家敗北

# 9. NPC AI 模型

## NPC A：激進擴張型（Profit Rush）

* 前期瘋狂擴張
* 中期排放爆炸，需要大量碳權

## NPC B：效率求極致型（Tech Optimizer）

* 優先升級工廠
* 高機率取得 A/B 低費率優惠

## NPC C：碳權投機商（Carbon Dealer）

* 買爆國內額度 → 高價賣出
* 幾乎不碰國外碳權

# 10. 使用者介面（UI/UX 架構）

## 主介面

* 上方：資源狀態（錢、排放、碳額度、工廠等級）
* 左側：地圖（可點擊土地）
* 右側：企業面板（工廠列表）
* 下方：行動選單

## 市場 UI（碳權交易）

* 左：國內額度（可買可賣）
* 右：國際碳權（只能買）
* 顯示價格波動、剩餘庫存

## 排放怪獸 UI

* 巨大陰影逐步逼近城市
* 下方顯示進度條（0～100%）

## 年度結算報表

包含：

* 本年收入
* 排放量細項
* 碳費計算式
* 地圖狀況
* 碳權扣減紀錄
* 排放怪獸成長率

# 11. 遊戲結束與計分

## 遊戲結束條件

* 10 回合後 → 進入總結算
* 排放怪獸到達城市 → 全敗

## 末局計算

1. 最高碳費者淘汰
2. 其餘玩家依下式計算：

```
Final Score = 營收 + 工廠資產 + 綠電加成 − CBAM 稅負
```
